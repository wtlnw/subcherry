What's New

Here is the description of significant changes and known issues for the current Subcherry release as well as for the previous ones in descending order of the release date.

Subcherry 1.9.1

This is a maintenance release containing the following bugfixes.

Bugfixes

  • Issue #16: Help contexts not included in the binary build.
  • Issue #17: Commit Messages are not rewritten properly when 'Port' mode was selected.

Known Issues

  • Issue #5: Support eclipse projects with different local and remote folder names.

Subcherry 1.9.0

The major focus of this release was providing built-in help for new users and minor usability improvements for the Subcherry Merge Wizard in order to minimize the replay setup process.

New Features and Improvements

  • Issue #6: Provide eclipse help.
  • Issue #7: Remove 'detect common modules' preference setting.
  • Issue #8: Remove 'detect common modules' preference setting.
  • Issue #9: Introduce confirmation dialogs for the 'skip' and 'reset' commands.
  • Issue #10: Introduce localization for literals.
  • Issue #11: Remove 'skip waiting for timestamp' preference setting.
  • Issue #12: Restrict selectable modules to same SVN repository location.
  • Issue #13: Prevent the wizard from finishing when no revisions were selected.
  • Issue #14: Check source branch against configured branch and trunk patterns.

Known Issues

  • Issue #5: Support eclipse projects with different local and remote folder names.
